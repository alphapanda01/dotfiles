### INSTALLATION

#### Vim installation
- install vim(ofc)
- clone and install VimPlug (https://github.com/junegunn/vim-plug)
- copy `.vimrc` to `$HOME`
- type `:PlugInstall` in vim cmd

#### shell
- Install oh-my-zsh(https://github.com/ohmyzsh/ohmyzsh)
- copy `.zshrc` to `$HOME`

#### SpaceVim (I use it with nvim)
- Install spacevim
- move `.SpaceVim.d` folder to `$HOME`
